lambda-lifting
==============

A visualizer for the Lambda Lifting game (ICFPC '12) in Haskell

Compiling & Running
===================

Build it:

    cabal configure
    cabal build

Then, you can start playing by:

    ./dist/build/Game/Game maps/contest1.map

Playing
=======

You can move the robot and play the game with the following keys:

    "i" is "Up"
    "k" is "Down"
    "j" is "Left"
    "l" is "Right"
    "w" is "Wait"
    "s" is "Play razor"
    "a" is "Abort"
